Photography Website

This project is a responsive photography portfolio website built using HTML5, CSS, Bootstrap, and JavaScript. The site showcases a photographer's work in a clean and visually appealing manner, with a user-friendly interface that adapts seamlessly to both mobile and desktop screens. The use of Bootstrap ensured fast development and a responsive grid system, while custom CSS was employed for additional styling. JavaScript was integrated to add dynamic features like image carousels and interactive navigation. The project is hosted on GitHub, allowing for easy access and collaboration.

Blog Post app

This blog post application is a full-stack web app built using **Express.js**, **EJS**, and **Bootstrap**. It allows users to create, edit, and view blog posts in a clean, responsive layout. **Express.js**, a fast and minimal web framework for **Node.js**, was used to handle the server-side logic, routing, and data management. For templating, **EJS** (Embedded JavaScript) was chosen to dynamically render the HTML pages with content from the server. **Bootstrap** was utilized to ensure the site looks great on both desktop and mobile screens, with a responsive grid system and predefined styling components. The application supports features such as adding new posts, editing existing ones, and deleting content, making it ideal for those looking to manage their blog efficiently. The combination of **JavaScript** on both the client and server side allows for seamless functionality and smooth navigation throughout the app.
